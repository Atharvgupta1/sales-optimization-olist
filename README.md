# 📊 Sales Optimisation Analysis (Olist E-commerce)

## 🚀 Project Overview

This project focuses on analysing e-commerce sales data to identify key revenue drivers, customer behaviour patterns, and operational inefficiencies. The goal is to provide actionable insights to improve business performance.

---

## 🎯 Business Problem

E-commerce businesses often struggle to:

* Identify high-performing products
* Retain valuable customers
* Manage delivery inefficiencies

This project addresses these challenges using data analytics.

---

## 🧰 Tech Stack

* Python (Pandas, NumPy)
* SQL (PostgreSQL)
* Power BI (Dashboard)
---

## 📂 Data Source

The dataset used in this project is the **Olist E-commerce Public Dataset**, which contains real-world transactional data from a Brazilian e-commerce platform.

* Source: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
* Includes: orders, customers, products, payments, and reviews data
  
This dataset enables analysis of sales performance, customer behaviour, and operational efficiency.

---

## 📊 Key Metrics

* Total Revenue: **16,643,731**
* Total Orders: **99,441**
* Avg Order Value: **167.37**
* Total Customers: **96,096**

---

## 🔍 Key Insights

* A small number of categories drive a large portion of revenue
* The top category **beleza_saude** generates ~1.49M in revenue
* Delivery delays significantly impact customer satisfaction:

* On-Time: **4.14**
* * Delayed: **2.32**
* Customer experience is strongly linked to operational performance

---

## 📈 Dashboard Overview

The Power BI dashboard includes:

* KPI summary (Revenue, Orders, AOV)
* Sales trend analysis
* Product & category performance
* Customer insights
* Payment analysis
* Delivery & review impact

---

## 💡 Business Recommendations

* Focus on high-performing categories
* Improve delivery logistics to reduce delays
* Increase average order value through bundling
* Retain high-value customers with targeted strategies

---

## 🧠 Conclusion

This project demonstrates how data analytics can uncover key insights that drive revenue growth and improve customer experience. By combining product, customer, and operational analysis, businesses can make more informed decisions.
